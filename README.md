# Web3 Daily 介绍

Web3 极客日报是为 Web3 时代的极客们准备的日常读物，由一群极客协作完成，每天更新，每期包含三个推荐内容，都来自极客们各自关注的领域。每晚由 Rebase 志愿者们整理发出。

## 参与方式一

- 通过微信群和催更表进行协作
- 包括 7 名催更人和若干名内容贡献者
  - 催更人负责整理内容
  - 贡献者每周贡献 1 个推荐

- 如果贡献者连续 3 周没有贡献，则会被取消贡献资格
